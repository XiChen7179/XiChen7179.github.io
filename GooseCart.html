    Personal Side Project

    GooseCart

    Description

    GooseCart is a cross-platform mobile app I co-founded and led as both PM and lead engineer through Georgia Tech’s Create-X. Designed to streamline group ordering in neighborhood communities, the app integrates with social platforms like WeChat and Facebook for seamless user onboarding and collaboration. I built the backend using Java Spring Boot, Firebase, and MongoDB, and shipped native frontends in Swift, Android, and React Native. On the AI side, I developed demand prediction and recommendation tools with Python and LangChain. GooseCart was named a top 5% Y Combinator Summer applicant and won support from the Johns Hopkins Incubator and Create-X Demo Day.

    Full Stack Developer

    As a full stack developer, I architect and implement scalable systems across the stack—from frontend interfaces in React/Swift to backend services in Java Spring Boot and Flask. My work emphasizes clean API design, modular architecture, and performance optimization across distributed systems. On the infrastructure side, I deploy and manage apps using Google Cloud (Cloud Functions, Firestore, Compute Engine), enabling real-time features, secure auth, and seamless CI/CD. I design with adaptability in mind—whether it’s cloud-based microservices or hybrid architectures that balance cost and speed.
