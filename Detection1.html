    ACT Lab: Innovative Cyberattack Detection Model

    CAN Detection with Adversarial Training

    Description

    The model is based on traditional CNN model but adding adversarial data to detect the anomality of datasets.

    Problem it solves

    In the Controller Area Network, or CAN, attackers can inject malicious messages, leading to system malfunctions or worse. Recognizing this high vulnerability, this research model introduces a novel intrusion detection system that leverages deep learning to safeguard vehicles against such threats.
